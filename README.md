# Fractals
A demonstration of classic fractals with a color palette generator in C

# Building
Clone this repository
```
git clone https://github.com/methusael13/fractals
```
Create a build directory and compile
```
cd fractals
mkdir build
cd build && cmake ..
make
```
Run example executables
```
bin/julia
```
